Extracts classes and label values of matching GTS.

```
[ 'token' 'class_selector' { labels_selectors } ] FINDSETS [ classes ] { 'label1' [ label1_values ] ... 'labeln' [ labeln_values ] } { 'attr1' [ attr1_values ] ... 'attrn' [ attrn_values ] }
```
