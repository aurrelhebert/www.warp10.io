The `TOKENINFO` function extracts information about the token on top of the stack.


## Example ##

WarpScript commands:

    'DOC-TOKEN'
    TOKENINFO

Stack (in JSON):

    TOP: {"application":"doc.data","issuance":0,"expiry":1220372016854776000,"type":"READ"}
